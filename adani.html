ROPE WAY-an eco friendly way of commuting

ROPE WAY- is a convenient mode to connect remote hilly areas and valleys with each other. It is an eco-friendly mode as one does not have to cut trees and destroy hill slopes to make way for roads on this terrain. It uses less fuel to move people from one point to another than a car,it can run on solar power and electric batteries,and is highly village friendly as there arent that moany cars there

PROS

  • It is easy to install
  • it doesnt harm the environment
  • It does not require roads
  • it can be solar powered
